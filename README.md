## How to use this template
1 - clone this repo 
```sh
git clone https://github.com/fernandollisboa/node-app-template.git
```
2 - install the dependencies
```sh
npm i
```
3 - fill all the .env files:
`.env`, `env.test` and `.env.dev`

4 - install husky
```sh
npx husky install
```
